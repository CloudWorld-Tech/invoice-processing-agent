# AI Coding Tool Transcripts

This folder contains raw interaction logs from AI-assisted development sessions used to build this project.

## Tool Used

- **Claude Code** (Anthropic CLI) — Claude Opus model
- Used for: architecture design, code generation, debugging, testing, and review

## How to Add Transcripts

Claude Code session transcripts are stored at:
```
~/.claude/projects/<project-hash>/<session-id>.jsonl
```

Copy relevant session files into this folder to include them in the deliverable.
